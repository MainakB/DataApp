Port Used - 8186



To start the app from terminal -

java -jar expeditors-dataapp-0.0.1-SNAPSHOT.jar



Command to run from browser (POSTMAN can be used as well for API calls)-


Descending order based on First Name -

http://localhost:8186/application/data?sortType=desc&name=First%20Name


Ascending order based on First Name -

http://localhost:8186/application/data?sortType=ascd&name=First%20Name


Descending order based on Last Name -

http://localhost:8186/application/data?sortType=desc&name=Last%20Name


Ascending order based on Last Name -

http://localhost:8186/application/data?sortType=ascd&name=Last%20Name
